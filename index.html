    2. Basic TagsðŸ‘‡

  2. Html tag: Creates an HTML document 
    <html></html>
  3. Head tag: Sets off the title & other info that isn't displayed 
    <head></head>
  4. Body tag: Sets off the visible portion of the document 
    <body></body>
  5. Title tag: Puts name of the document in the title bar; when bookmarking pages, this is what is bookmarked 
    <title></title>

    6. Body attributes (only used in email newsletters)ðŸ‘‡

  6. Body bgcolor: Sets background color, using name or hex value 
    <body bgcolor=?>
  7. Body text: Sets text color, using name or hex value 
    <body text=?>
  8. Body link: Sets color of links, using name or hex value 
    <body link=?>
  9. Body vlink:Sets color of visited links, using name or hex value 
    <body vlink=?>
  10. Body alink: Sets color of active links (while mouse-clicking) 
    <body alink=?>

    11. Page layout

    Note- It is very important!
  11. Header tag: It used to add navbar content and headings of the page. 
    <header></header>
  12. Main tag: It is used as a main content of the webpage 
    <main></main>
  13. Footer tag: It used as a bottom content of the webpage 
    <footer></footer>
  14. Section tag: It used divide the code in section (used in inside the main tag) 
    <section></section>
  15. Article tag: It is used to add some important text 
    <article></article>
  16. Aside tag: Content aside from the content(eg-adds etc) 
    <aside></aside>

    17. FORMATTING ðŸ‘‡

  17. Paragraph tag : Creates a new Paragraph 
    <p>//contains paragraph</p>
  18. Br tag: A Inserts a line break 
    <br></br>
  19. Blockquote tag: Puts content in a quote - indents text from both sides 
    <blockquote></blockquote>
  20. Div tag: Used to format block content with CSS 
    <div></div>
  21. Span tag: Used to format inline content with CSS 
    <span></span>

    22. LISTS ðŸ‘‡

  22. Ul tag: Creates an unordered list 
    <ul></ul>
  23. Ol tag: Creates an ordered list (start=xx, where xx is a counting number) 
    <ol start=?></ol>
  24. Li tag: Encompasses each list item 
    <li></li>
  25. Dl tag: Creates a definition list 
    <dl></dl>
  26. Dt tag: Precedes eachdefintion term 
    <dt></dt>
  27. Dd tag: Precedes eachdefintion 
    <dd></dd>

    28. Text TagsðŸ‘‡

  28. Pre tag: Creates preformatted text 
    <pre></pre>
  29. Heading tags: Creates headlines -- H1=largest, H6=smallest 
    <h1></h1>
    <h2></h2>
    <h3></h3>
    <h4></h4>
    <h5></h5>
    <h6></h6>
  30. Bold tag: Creates bold text (should use <strong> instead) 
    <b></b>
  31. Underline tag: Creates underline on text 
    <u></u>
  32. Italic tag: Creates italicized text (should use <em> instead) 
    <i></i>
  33. Tt tag: Creates typewriter-style text 
    <tt></tt>
  34. code tag: Used to define source code, usually monospace 
    <code></code>
  35. Cite tag: Creates a citation, usually processed in italics 
    <cite></cite>
  36. Address tag: Creates address section, usually processed in italics 
    <address></address>
  37. Em tag: Emphasizes a word (usually processed in italics) 
    <em></em>
  38. Strong tag: Emphasizes a word (usually processed in bold) 
    <strong></strong>
  39. Font size tag: Sets size of font - 1 to 7 (should use CSS instead) 
    <font size=?> </font>
  40. Font color tag: Sets font color (should use CSS instead) 
    <font color=?> </font>
  41. Font face tag: Defines the font used (should use CSS instead) 
    <font face=?> </font>

    42. Graphical elementsðŸ‘‡

  42. Hr tag: Inserts a horizontal rule 
    <hr>
  43. Hr size tag: Sets size (height) of horizontal rule 
    <hr size=?>
  44. Hr width tag: Sets width of rule (as a % or absolute pixel length) 
    <hr width=?>
  45. Hr noshade tag: Creates a horizontal rule without a shadow 
    <hr noshade>
  46. Image tag: Adds image; it is a separate file located at the URL 
    <img src="<!--url of the image-->"/>
  47. Image tag with align attribute: Aligns image left/right/center/bottom/top/middle (use CSS) 
    <img src="<!-- Url of the image-->" align=?>
  48. Image tag with border attribute: Sets size of border surrounding image (use CSS) 
    <img src="<!--url of the image" border=?>
  49. Image tag with height attribute: Sets height of image, in pixels 
    <img src="<!--url of the image-->" height=?>
  50. Image tag with width attribute: Sets width of image, in pixels 
    <img src="<!-- url of the image-->" width=?>
  51. Image tag with alt attribute: Sets the alternate text for browsers that can't process images (required by the ADA) 
    <img src="<!--url of the image-->" alt=?>
  52. Button tag: It help to create clickable button in HTML 
    <button type="button"> </button>

    53. LINKSðŸ‘‡

  53. Anchor tag: Creates a hyperlink to a Uniform Resource Locator 
    <a href="URL">clickable text</a>
  54. Email anchor tag: Creates a hyperlink to an email address 
    <a href="mailto:EMAIL_ADDRESS">clickable text</a>
  55. Anchor tag with name attribute: Creates a target location within a document 
    <a name="NAME">
  56. Anchor tag with href attribute: Creates a link to that target location 
    <a href="#NAME">clickable text</a>

    57. FormsðŸ‘‡

  57. Form tag: Defines a form 
    <form></form>
  58. Select tag with multiple name attribute: Creates a scrolling menu. Size sets the number of menu items visible before user needs to scroll. 
    <select multiple name=? size=?> </select>
  59. Select tag with name attribute: Creates a pulldown menu 
    <select name=?></select>
  60. Option tag: Sets off each menu item 
    <option>
  61. Textarea tag: Creates a text box area. Columns set the width; rows set the height. 
    <textarea name=? cols="x" rows="y"></textarea>
  62. Input checkbox tag: Creates a checkbox. 
    <input type="checkbox" name=? value=?>
  63. Input checkbox(checked) tag: Creates a checkbox which is pre-checked. 
    <input type="checkbox" name=? value=? checked>
  64. Input radio tag: Creates a radio button. 
    <input type="radio" name=? value=?>
  65. Input radio tag(checked): Creates a radio button which is pre-checked. 
    <input type="radio" name=? value=? checked>
  66. Input image tag: Creates a one-line text area. Size sets length, in characters. 
    <input type="text" name=? size=?>
  67. Input submit tag: Creates a submit button. Value sets the text in the submit button. 
    <input type="submit" value=?>
  68. Input image tag: Creates a submit button using an image. 
    <input type="image" name=? src=? border=? alt=?>
  69. Input reset tag: Creates a reset button 
    <input type="reset">

    70. Tables (use only for data layout - use CSS for page layout) ðŸ‘‡

  70. Table tag: Creates a table 
    <table></table>
  71. Table row tag: Sets off each row in a table 
    <tr></tr>
  72. Table data tag: Sets off each cell in a row 
    <td></td>
  73. Table heading 
    <th></th>

    74. HTML5 input tag attributes ðŸ‘‡

    Notes- not all this browsers support; visit http://caniuse.com for details
  74. Input email tag: Sets a single-line textbox for email addresses 
    <input type="email" name=?>
  75. Input url tag: Sets a single-line textbox for URLs 
    <input type="url" name=?>
  76. Input number tag: Sets a single-line textbox for URLs 
    <input type="number" name=?>
  77. Input range tag: Sets a single-line text box for a range of numbers 
    <input type="range" name=?>
  78. Input date/month/week/time tag: Sets a single-line text box with a calendar showing the date/month/week/time 
    <input type="date/month/week/time" name=?>
  79. Input search tag: Sets a single-line text box for searching 
    <input type="search" name=?>
  80. Input color: Sets a single-line text box for picking a color 
    <input type="color" name=?>

    81. Table attributes (only use for email newsletters)

  81. Table border tag: Sets the width of the border around table cells 
    <table border=?>
  82. Table cellspacing tag: Sets amount of space between table cells 
    <table cellspacing=?>
  83. Table cellpadding tag: Sets amount of space between a cell's border and its contents 
    <table cellpadding=?>
  84. Table width tag: Sets width of the table in pixels or as a percentage 
    <table width=?>
  85. Table row align tag: Sets alignment for cells within the row (left/center/right) 
    <tr align=?>
  86. Table data align tag: Sets alignment for cells (left/center/right) 
    <td align=?>
  87. Table row valign tag: Sets vertical alignment for cells within the row (top/middle/bottom) 
    <tr valign=?>
  88. Table data valign tag: Sets vertical alignment for cell (top/middle/bottom) 
    <td valign=?>
  89. Table data rowspan: Sets number of rows a cell should span (default=1) 
    <td rowspan=?>
  90. Table data colspan: Sets number of columns a cell should span 
    <td colspan=?>
  91. Table data nowrap: Prevents lines within a cell from being broken to fit 
    <td nowrap>
